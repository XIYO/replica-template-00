# 내 몸 이해하기: 건강 지표 해석

건강 관리는 막연한 노력이 아니라, **내 몸에 대한 정확한 이해**에서 시작됩니다. 기본적인 건강 지표를 이해하고 이를 통해 자신의 건강 상태를 파악하는 것은 건강 관리의 첫걸음입니다. 이 문서에서는 주요 건강 지표를 설명하고, 스스로 측정하고 해석하는 방법을 안내합니다.

## 주요 건강 지표

*   **혈압:** 혈액이 혈관 벽을 міші는 힘을 나타냅니다. 수축기 혈압(최고 혈압)과 이완기 혈압(최저 혈압)으로 구성됩니다. 정상 혈압은 120/80mmHg 미만입니다. 고혈압은 심혈관 질환의 위험을 높입니다.

    ```
    # 혈압 측정 예시
    수축기 혈압: 130mmHg
    이완기 혈압: 85mmHg
    # 결과: 고혈압 전단계
    ```

*   **심박수:** 1분 동안 심장이 뛰는 횟수를 나타냅니다. 안정 시 심박수는 일반적으로 60~100회/분입니다. 운동을 하면 심박수가 증가합니다.

    ```
    # 심박수 측정 예시
    안정 시 심박수: 70회/분
    # 결과: 정상
    ```

*   **체온:** 몸의 온도를 나타냅니다. 정상 체온은 36.5~37.5°C입니다. 체온이 높으면 발열을 의미할 수 있습니다.

*   **체질량지수(BMI):** 키와 몸무게를 이용하여 체지방량을 추정하는 지표입니다. BMI = 몸무게(kg) / (키(m))^2 입니다. 18.5~22.9는 정상, 23~24.9는 과체중, 25 이상은 비만으로 분류됩니다.

    ```
    # BMI 계산 예시
    키: 1.75m
    몸무게: 75kg
    BMI = 75 / (1.75 * 1.75) = 24.49
    # 결과: 과체중
    ```

*   **혈당:** 혈액 속에 포함된 포도당의 농도를 나타냅니다. 공복 혈당은 100mg/dL 미만이 정상입니다. 126mg/dL 이상이면 당뇨병으로 진단될 수 있습니다.

*   **콜레스테롤:** 혈액 속에 존재하는 지방의 일종입니다. LDL 콜레스테롤(나쁜 콜레스테롤)과 HDL 콜레스테롤(좋은 콜레스테롤)로 나뉩니다. LDL 콜레스테롤은 낮을수록, HDL 콜레스테롤은 높을수록 좋습니다.

## 건강 지표 측정 및 관리

위에서 언급한 건강 지표들은 가정용 혈압계, 체온계, 체중계 등을 이용하여 비교적 쉽게 측정할 수 있습니다. 혈당 및 콜레스테롤 수치는 병원에서 혈액 검사를 통해 확인할 수 있습니다. 꾸준히 건강 지표를 측정하고 기록하여 변화를 추적하고, 이상 징후가 발견되면 의사와 상담하는 것이 중요합니다.
