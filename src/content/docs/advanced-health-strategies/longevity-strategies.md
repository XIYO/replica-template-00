# 장수 전략: 건강하게 오래 사는 법

수명 연장은 인류의 오랜 염원입니다. 단순히 오래 사는 것뿐만 아니라 건강하게, 활기차게 오래 사는 것이 더욱 중요합니다. 이 문서에서는 건강하게 오래 살기 위한 다양한 전략과 최신 연구 결과를 소개합니다.

## 식습관

*   **칼로리 제한:** 칼로리 섭취를 줄이는 것은 수명 연장과 관련된 가장 잘 연구된 방법 중 하나입니다. 적절한 칼로리 제한은 세포 손상을 줄이고 대사 건강을 개선할 수 있습니다.

    ```python
    # 예시: 하루 섭취 칼로리 계산 및 기록
    def calculate_daily_calories(weight, activity_level):
        # 기초 대사량 (BMR) 계산 - 해리스-베네딕트 방정식 사용 (간단하게)
        bmr = 10 * weight  # kg 단위
        
        # 활동 수준에 따른 칼로리 요구량 계산
        if activity_level == "sedentary":
            calories = bmr * 1.2
        elif activity_level == "lightly_active":
            calories = bmr * 1.375
        elif activity_level == "moderately_active":
            calories = bmr * 1.55
        else:
            calories = bmr * 1.725
        
        return calories

    weight = 70  # kg
    activity_level = "moderately_active"
    daily_calories = calculate_daily_calories(weight, activity_level)
    print(f"하루 권장 칼로리 섭취량: {daily_calories:.2f} kcal")
    ```

*   **지중해식 식단:** 과일, 채소, 통곡물, 올리브 오일, 생선 등을 중심으로 한 지중해식 식단은 심혈관 질환, 암, 치매 등 다양한 질병의 위험을 줄여줍니다. [건강한 식습관 만들기: 균형 잡힌 식단 가이드] 문서를 참고하여 식단을 구성해보세요.
